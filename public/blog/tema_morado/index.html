Tema morado oscuro para RStudio

July 23, 2024

código

base16 purple dark

Tema para RStudio enfocado en una paleta de colores morada y rosada, basado en el tema base16 Default Dark de {rsthemes}, which in turn is based on base16.

Instalación

Para agregar el tema a RStudio, descarga el archivo .rstheme desde el repositorio, luego abre las opciones globales de RStudio Global Options (⌘;), entra al menú Appareance, presiona Add… en la parte de abajo, y elige el archivo base16-purple-dark.rstheme.
